# pinenut
A tiny rofi based music selector for mpv

```
Pinenut v1.0 - A tiny rofi based music selector for mpv
Usage : pinenut [options]

Options :
 -m		play selected track from playlist
 -e		outputs the currently playing track
 ```

# Connect to an api
